# 欧美玩具认证指南（钩织毛线玩具）

## 🇺🇸 美国 — CPC 认证

**监管机构**：CPSC（消费者产品安全委员会）
**适用**：12岁以下儿童产品
**文件**：CPC（Children's Product Certificate）+ 第三方检测报告

### 检测项目（ASTM F963）

| 项目 | 跟钩织玩具的关系 |
|------|-----------------|
| 铅含量 | 毛线染料可能含铅，必测 |
| 邻苯二甲酸盐 | 塑料配件（眼睛/鼻子/填充物） |
| 小零件 | 纽扣眼睛脱落→窒息风险→**换刺绣** |
| 阻燃性 | 毛线是纺织品，必测 |
| 机械物理 | 缝合强度、拉扯测试 |

### 操作流程
1. 找 CPSC 认可实验室（SGS/BV/Intertek/TÜV，中国有）
2. 寄样 3-5 件
3. 拿检测报告后自己出具 CPC 证书（模板在 CPSC 官网）
4. 每批货抽查留底

费用：$200-800/产品类别 | 周期：2-4周

**省钱技巧**：同批毛线做多款→按材质认证，不用每款都测。

## 🇪🇺 欧盟 — CE + EN71

**适用**：所有玩具

### 检测项目

| 标准 | 内容 |
|------|------|
| EN71-1 | 机械物理安全（缝合强度、小零件） |
| EN71-2 | 阻燃性 |
| EN71-3 | 19种重金属迁移量 |
| REACH | 化学品注册（染料、填充物成分） |

### 额外文件
- **技术文件**（Technical File）：产品描述、设计图、材料清单、检测报告
- **符合性声明**（DoC）：自己签署的法律文件
- **欧代**（EC Rep）：非欧盟公司必须有，年费 €100-300

费用：€300-1500（检测） + €100-300/年（欧代） | 周期：3-6周

---

## ⚡ 省钱省事方案

| 策略 | 做法 |
|------|------|
| 中国本地实验室 | SGS/BV/TÜV 在国内有分公司，便宜 60% |
| 按材质认证 | 同材料同填充物=一次检测覆盖全系 |
| 用刺绣代替塑料眼 | 省小零件测试，更安全 |
| 先 EU 后 US | EN71 比 CPC 容易过 |
| 标签标 3+ 岁 | 3岁以下要求严很多，先避开 |

---

## 实操清单

1. 确定目标市场 → 选定认证标准
2. 整理材料清单（毛线品牌+成分、填充物、配件）
3. 联系 SGS/BV/TÜV 询价寄样
4. 等报告（2-4周）
5. EU 产品找欧代（淘宝搜"欧盟授权代表"，几百块）
6. 出 CPC/DoC 文件
7. CE 标贴产品 + 包装印警告语
