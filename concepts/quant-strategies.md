# 资本保全型量化策略研究

> 📅 2026-06-13 | 目标：2万本金不亏损，收益尽量高

## 核心结论：没有"保本高收益"的策略

交易铁律：收益和风险同源。任何能产生超额收益的策略都会有回撤。

| 目标 | 现实 |
|------|------|
| 本金**完全**不亏 | 做不到 — 最好的策略也有15-20%回撤 |
| 收益尽量高 | 可以 — 通过回撤控制和仓位管理接近目标 |

---

## 推荐方案：金字塔风险结构

```
┌──────────────┐
│ 进攻层 40%   │ → NostalgiaForInfinity X6（+42%/6月，回撤18.7%）
├──────────────┤
│ 防守层 35%   │ → 自定义保守策略（只做BTC/ETH，3-5%止损，1-2%止盈）
├──────────────┤
│ 现金流 25%   │ → 持有USDT，等待极端机会（大跌抄底）
└──────────────┘
```

### 具体参数

**进攻层（$8,000）**
策略：NostalgiaForInfinity X6
- 6个月实测：+42.3% 收益，最大回撤 18.7%，胜率 58%
- 6 个同时开仓（不超 8 个）
- 只做 USDT 交易对
- 只做币安前十流动性币种
- derisking system 自动在市场恶化时减仓

**防守层（$7,000）**
策略：BTC/ETH 保守网格
- 只在 BTC/ETH 两个币种交易
- 单笔止损 3%，止盈 2%
- RSI 超卖 (>30) 进场
- 同时最多 3 个仓位
- 期望：月收益 3-8%，回撤 < 10%

**现金流（$5,000）**
策略：等机会
- 持有 USDT（无风险）
- 触发条件：BTC 单日跌 >15% → 买入
- 或市场极度恐慌（Fear & Greed < 15）→ 买入

## 为什么不是 5 倍杠杆

用 5 倍杠杆做到 3 个月 500% 需要两个条件同时满足：
1. 市场单向趋势（不能震荡，震荡中杠杆磨损巨大）
2. 策略 100% 正确率（但最优策略也只有 58%）

杠杆 5 倍 + 胜率 58% + 3 个月 = 期望结果：**爆仓**
