# 沙特阿拉伯外贸深度指南

> 最后更新：2026-06-13 | 调研来源：商务部、海关总署、行业报告

## 一、市场总览

| 指标 | 数据 |
|------|------|
| 人口 | 3,500万+，70% 在 35 岁以下 |
| GDP | $1.24万亿（人均 $35,000） |
| 中国对沙出口 | $534亿（2025），+6.7% |
| 中国占比 | 沙特最大进口来源国，占 21% |
| 关税政策 | 低关税、自由贸易，已取消全部出口海关服务费 |

**核心驱动力：2030 愿景**
- NEOM 新城（消耗全球 20% 钢材）
- 红海旅游项目、2034 世界杯场馆
- 非油出口从 16% 提升到 50%
- 政府非油收入从 $435亿 提升到 $2,667亿

## 二、合规与认证（最大门槛）

### SABER 认证体系（强制）

所有出口沙特的产品必须通过 SABER 平台注册：

1. **产品证书**（有效期 1 年）→ 产品注册
2. **批次证书**（每票货）→ 发货前申请
3. **HS 编码必须精确** → 错一个数字延迟 2-3 周
4. **认证机构**：Intertek、Bureau Veritas，检测周期 10-14 天

⚠️ **42%的海关扣押是因为缺 SABER 二维码**

### 必备单证清单

| 单证 | 要求 |
|------|------|
| 商业发票 | **双语**（阿拉伯语+英语），含 15% VAT |
| 提单 | 海运提单或空运提单 |
| 原产地证 | 中国贸促会签发 |
| SABER 证书 | 产品证书 + 批次证书 |
| 特定行业认证 | 食品需 SFDA，电子产品需 GCC 标志 |

### 包装标签要求

- 食品/化妆品：阿拉伯语有效期，格式 `YYYY/MM/DD`
- **禁用绿色**（伊斯兰专属色）
- **禁用十字架符号**
- 玩具/卡通图案需符合当地文化价值观（有 7,000 件玩具因卡通图案被拒案例）

## 三、物流通道

### 港口选择

| 港口 | 适合品类 | 航程 | 备注 |
|------|------|:---:|------|
| **达曼** (Dammam) | 工业设备、原材料 | 18-25天 | 宁波→达曼 18天快线 |
| **吉达** (Jeddah) | 消费品→利雅得 | 22-30天 | 旺季拥堵 |

### 海运时效与成本

| 方式 | 时效 | 成本 | 适合 |
|------|:---:|------|------|
| 整柜 FCL | 22-30天 | 低 | 大批量 |
| 拼箱 LCL | 25-35天 | 中 | 小批量 |
| 空运 | 4-7天 | 高 | 急货/高值 |
| 门到门 DDP | 28-40天 | 中高 | 新手首选 |

### 河北出口路线

```
石家庄 → 天津港（300km，陆运半天）
天津港 → 达曼港（海运 20-25 天）
达曼 → 利雅得（陆运 4-6 小时）
```

> 朝觐季（6-11月）价格涨 15-30%，建议 1-4 月发货避峰

## 四、商务文化与谈判

### 核心原则

| 原则 | 说明 |
|------|------|
| **关系先行** (Wasta) | 类似中国的"关系"，先做人再做事 |
| **慢节奏** | 着急 = 不尊重。谈判可能多轮，耐心是美德 |
| **等级制** | 决策权在最高层，绕过层级是冒犯 |
| **面对面** | 邮件不够，必须电话或见面跟进 |
| **斋月** | 斋月期间商务停滞，提前排期 |

### 见面礼仪

| 场景 | 做法 |
|------|------|
| 问候 | `Assalamu alaikum`（愿平安与你同在） |
| 握手 | 从右到左，**不和异性握手除非对方先伸手** |
| 名片 | 右手递接 |
| 穿着 | 男士西装领带，女士穿 abaya + 带头巾 |
| 送礼 | 高品质但不含酒精/猪皮制品 |

### 商务时间

- 工作日：周日至周四（**周五周六休息**）
- 斋月：工作时间缩短至 6 小时
- 每天 5 次祷告时间，商务活动暂停

## 五、付款与金融

| 方式 | 风险 | 建议 |
|------|:---:|------|
| TT 30% 定金 + 70% 发货前 | 低 | ✅ 首选 |
| 信用证 (LC) | 中 | 大单用，手续繁琐 |
| 赊账 (OA) | 高 | ❌ 新手避免 |

**注意**：合同不得含延迟付款利息条款（违反伊斯兰金融法），改签"折扣"或"管理费用"。

## 六、风险清单

| 风险 | 级别 | 对策 |
|------|:---:|------|
| SABER 认证不通过 | 🔴 高 | 发货前拿到证书，用有经验的认证代理 |
| 海关扣押 | 🟡 中 | HS 编码反复核对，单据双语完整 |
| 付款拖欠 | 🟡 中 | 坚持发货前付清，新客户不赊账 |
| 文化冒犯 | 🟢 低 | 尊重伊斯兰习俗，找当地代理 |
| 反倾销税 | 🟡 中 | 发货前查海关编码对应税率（钢管类已有关税） |
| 物流延误 | 🟡 中 | 避开朝觐季，预留 2 周缓冲 |
| 政治/区域冲突 | 🟢 低 | 红海航运偶受干扰，走达曼更稳定 |

## 八、执行路线图（新手版）

```
第 1 步 ── 产品验证（2周）
  ├── 确认选品，找到河北工厂
  ├── 拿样品，确认出厂价
  └── 查询沙特对应 HS 编码及关税

第 2 步 ── SABER 认证（3-4周）
  ├── 找认证代理（Intertek/Bureau Veritas）
  ├── 样品送检，获取测试报告
  └── 在 SABER 平台注册产品证书

第 3 步 ── 开发客户（并行进行）
  ├── 阿里国际站 / 中国制造网 开店
  ├── LinkedIn 加沙特买家
  ├── 参加沙特五大行业展（Saudi Big 5）
  └── 找迪拜/沙特当地华人贸易代理

第 4 步 ── 试单（1-2个月）
  ├── 谈成第一个小单（1-2个柜）
  ├── 天津港发货，走 DDP 门到门
  ├── 准备全套单证 + SABER 批次证书
  └── 跟踪到港+清关+送达

第 5 步 ── 复盘放量
  ├── 总结首次问题
  ├── 跟客户建立长期关系
  └── 每年参加 1-2 次中东展会
```
